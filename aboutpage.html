About this project

This is a small personality-test web project designed for a beginner web assignment. It demonstrates:

  • Semantic HTML structure and accessible controls
  • External CSS with Grid/Flexbox layout
  • Client-side validation for a simple contact form

Services / Features

  1. Personality Quiz: 15 questions, quick summary.
  2. Responsive design: mobile-first and touch friendly.
  3. Clear code: works well for reading and grading.

Contact

Use this simple contact form — it validates the fields client-side and will show error messages if something's wrong. (There is no server-side submission in this demo.)
